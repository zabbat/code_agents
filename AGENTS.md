# Agents

This project is dedicated to exploring and experimenting with:

- **GitHub Actions** — workflows, automation, CI/CD pipelines
- **GitHub Copilot Code Agent** — AI-assisted coding, autonomous PR generation, agentic workflows
- **Custom Agents** — building and integrating custom AI agents
- **Other agent frameworks and tools** — exploring the broader agentic ecosystem

## Tools

- **GitHub Copilot Pro** — primary AI assistant used throughout this project
